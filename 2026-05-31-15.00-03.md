## Senior Engineer Netflix Bikin Project Headroom: Aplikasi Open Source yang Bisa Potong Tagihan AI 90%!

Tagihan AI bikin pusing? Senior engineer Netflix, Tejas Chopra, ngerasain hal yang sama setelah dapat tagihan $287 dari Claude Sonnet cuma untuk debugging dan refactoring. Dari situ, Chopra bikin Project Headroom — software open source yang kompres token sebelum dikirim ke LLM, dan hasilnya? Bisa ngurangin token yang dipake sampai 90%!

Chopra menemukan bahwa sebagian besar token yang masuk ke LLM adalah data redundan: JSON skema yang verbose, metadata mesin, template API, kolom database identik. Bukan instruksi dari developer. Makanya Headroom dirancang untuk ngompres semua material sumber — riwayat percakapan, log, output tools, file, dan dokumen RAG — sebelum nyampe ke context window.

Hasilnya? Headroom yang baru rilis sejak Januari udah ngumpulin 2.000 bintang di GitHub, dipakai oleh banyak tim di Netflix, dan secara total udah menghemat sekitar $700.000 untuk para penggunanya. 200 miliar token berhasil dihemat! Dengan model pricing yang makin mahal ($3 per juta input token buat Claude Sonnet), tools kayak Headroom jadi penyelamat dompet para developer AI di mana-mana.
