## "What a joke": Sistem Billing Token GitHub Copilot Bikin Developer Naik Darah

GitHub baru aja ngumumin perubahan sistem pricing buat Copilot, dari langganan flat jadi token-based billing. Dan reaksi developer? Kurang lebih kayak orang dikasih tahu harga kopi naik 3x lipat — marah besar.

Di artikel TechCrunch yang bikin heboh ini, para developer ngeluh keras karena sistem token bikin biaya Coding Assistant mereka gak bisa diprediksi. Dulu bayar flat $10-$19 per bulan, sekarang harus hitung-hitungan token kayak maen game gacha. "The golden age of Microsoft's Github Copilot appears to be at an end," tulis TechCrunch.

Masalah utamanya: developer gak tahu berapa banyak token yang bakal mereka pake tiap bulan. Satu session coding ringan bisa abis puluhan ribu token. Kalo lagi lembur ngerjain bug? Bisa boncos. Model kayak gini bikin developer dan tim kecil paling kena dampak, karena biaya coding assistant bisa naik drastis tanpa peringatan.

Yang ironis, GitHub Copilot selama ini jadi salah satu produk AI coding paling populer di dunia. Tapi perubahan pricing ini bisa bikin banyak developer cabut ke alternatif kayak Cursor, Codeium, atau Claude yang masih pake model flat fee.
